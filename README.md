# BSc Thesis – Cancer Hallmarks in RNA-seq Data

This repository contains my BSc thesis from DTU Health Tech.

## Title
**Enabling Analysis of Cancer Hallmarks in RNA-sequencing Data**

## Summary
This project focused on developing and validating a cancer hallmark-associated gene set collection and applying it to RNA-seq breast cancer data from TCGA.

The work included:
- RNA-seq based analysis
- gene set collection development
- hallmark-oriented cancer analysis
- clustering and enrichment analysis
- transcriptomic interpretation in cancer subtypes

## What this demonstrates
This project reflects my profile in:
- transcriptomics
- RNA-seq analysis
- cancer biology
- enrichment analysis
- biological interpretation of large-scale data
